Coaching Institute Website (Three.js)

A modern coaching institute landing page built using HTML, CSS, JavaScript, and Three.js.
This project focuses on creating an interactive UI with a 3D animated hero section.

✨ Features:

🎓 Modern landing page design
🌌 3D animated hero section (Three.js)
📚 Floating 3D elements (books, cap, particles)
🖱️ Mouse interaction with camera movement
📱 Basic responsive layout
🎨 Clean and minimal UI
🛠️ Tech Stack
HTML5
CSS3
JavaScript (ES6)
Three.js
